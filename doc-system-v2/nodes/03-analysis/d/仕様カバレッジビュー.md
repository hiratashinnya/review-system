**もの**: D-4 から射影した仕様カバレッジ計測用スライス（FR・SPEC・TD ノードと condition 属性・refines 辺の集合）
**種別**: 内部データ（系外へ出ない）
**生成元**: P-1-6（検査ビュー射影）に依存（D→P）
**消費先**: P-3-2-1（FR×condition 充足集計）が P→D-21 で消費
**含むフィールド**: FR ノード（id・edges.to）・SPEC ノード（id・condition・edges.to で refines 先 FR を特定）・TD ノード（id・condition・edges.to で verifies 先 SPEC を特定）

**スタンプ結合解消**: 分析層トポロジ・接続検査・属性検査の情報を受けない。カバレッジ計測が必要とする FR/SPEC/TD の condition 軸情報だけの最小集合。旧 D-4 では condition が定義に含まれず PR6 断絶があったが D-21 の condition 含有で解消。
