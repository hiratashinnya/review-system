**前提条件**: 型が PORT（ポート）のノードが存在する（implementation ステージ到達済み・SRC 著作後）
**入力/トリガ**: PORT が SRC から被依存辺を受けていない（`must_be_linked_from: PORT←[SRC]`）。ただし充足に数えられるのは設計種別 PORT に適格なシンボル種別を持つ SRC のみ（シンボル適格性条件・`src.kind`×設計種別マッチング・詳細は別 DD で機械検証可能化・オーナー再確認待ち）
**期待動作**: RULE-006 ERROR を報告する（DD-9・設計ノードは実装 SRC で実現される＝価値経路の下流連続性）
**例**: PORT-1 を指す適格な SRC の辺が 1 本も無い → `ERROR|...|RULE-006|PORT-1|...`
