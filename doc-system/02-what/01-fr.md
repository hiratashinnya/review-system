# 機能要求

> **型**: FR ／ **必須上流**: SR（refines ✅）
> RULE-017: この FR を refines する SPEC の中に `condition: normal` が必要。

---

<details><summary>⬡ FR-1 · v0.1</summary>

```yaml

  - to: SR-1
```

## 自動修正の適用と revert

<details><summary>⬡ FR-7 · v0.1</summary>

```yaml

type: FR
labels: []
scheduled: ""
edges:
  - to: SR-2
    kind: refines
    status: pending
    ref_version: "0.1"
  - to: SR-4

    status: pending
    ref_version: "0.1"
```
</details>

決定的ツール範囲の 🤖 指摘を自動適用（O-3）し、finding 単位コミットで個別・一括 revert（O-6）を可能にする。
上流（④評価/⑤検証）がクリーン完了した場合のみ適用を走らせる（半端コミットを残さない）。
