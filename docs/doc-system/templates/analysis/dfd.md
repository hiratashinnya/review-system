# DFD・論理プロセス

> **型**: P ／ **必須上流**: SPEC（依存辺 ✅）
> **必須(§10)**: I/O/E との依存辺が1本以上
> 方向: **P → I**（プロセスは消費する入力に依存）／ **O → P**（出力は生成プロセスに依存・O 側から張る）／ **P → E**（プロセスはトリガ事象に依存）。
> 内部データフローは新型 **D**（**D → P**・系外に出ない）。
> プロセス間はI/O/Dノードを介して繋ぐ。P 同士を直接リンクしない（03 §7）。
> 階層は ID パターン `X-N` から推論（decomposes 廃止）。
